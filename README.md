# stock-management

a stock management applcation that allows user to :

  
	_display products that are saved in the application

	_add a product

	_update a product
  
used technologies :

	server side : nodejs 

	plateform : expressjs 

	database : Mongodb 

	Object data model(ODM) : Mongoose 

	veiw engine : EJS 

	other technologies : passportjs
